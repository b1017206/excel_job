# excel_job
エクセルファイルから学生のアンケート情報を出力する簡易バージョン
# 手順
*サンプルデータ（202004_前期.xlsx、201909_後期.xlsx）とexcel_job.ipynbを好きなディレクトリに入れる

*excel_job.ipynbをターミナル（コマンドプロンプト）で
```bash
jupyter notebook
```
と打ち、起動

*excel_job.ipynbソース内に入り、一つずつ実行

*最後まで行くと、graph.pngとout.xlsxが出力されている
